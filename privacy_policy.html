1. Información General

Esta política de privacidad describe cómo la aplicación Lyrion ("la aplicación", "nosotros") recopila, usa y protege la información cuando utilizas nuestra aplicación de chatbot con inteligencia artificial.

🔒 Transparencia sobre el Permiso de Cámara

¿Por qué necesitamos acceso a la cámara?

La aplicación solicita permiso de cámara (android.permission.CAMERA) ÚNICAMENTE para la función de OCR (reconocimiento de texto en imágenes). Esta función te permite:

  • Fotografiar documentos, pantallas o cualquier texto impreso
  • Extraer automáticamente el texto de la imagen
  • Usar ese texto en tus conversaciones con la IA

¿Qué NO hacemos con tu cámara?

  • ❌ NO tomamos fotos sin tu conocimiento
  • ❌ NO guardamos las imágenes en tu dispositivo
  • ❌ NO enviamos imágenes a servidores externos
  • ❌ NO accedemos a tu galería de fotos
  • ❌ NO utilizamos la cámara para ningún otro propósito

Control total: Puedes denegar el permiso de cámara y la aplicación funcionará perfectamente sin la función de OCR.

2. Datos que Recopilamos

2.1 Datos que SÍ recopilamos:

  • Conversaciones de chat: Los mensajes que intercambias con el chatbot se almacenan localmente en tu dispositivo usando ObjectBox (base de datos local)
  • Configuraciones de la aplicación: Preferencias de tema, modelo seleccionado, configuraciones de la interfaz y parámetros de IA (temperatura, tokens, etc.)
  • Reportes de contenido: Cuando reportas contenido inapropiado, almacenamos el reporte localmente para mejorar la moderación
  • Datos de archivo: Cuando utilizas funciones como OCR o análisis de imágenes, los archivos se procesan localmente
  • Imágenes seleccionadas de galería: Cuando seleccionas imágenes de tu galería para procesamiento OCR, solo accedemos a las imágenes específicas que eliges. Estas se procesan inmediatamente en tu dispositivo para extraer texto y NO se almacenan permanentemente ni se copian a otra ubicación
  • Imágenes de cámara para OCR: Cuando utilizas la función de OCR (reconocimiento óptico de caracteres), la aplicación solicita acceso a la cámara para capturar fotografías de texto. Estas imágenes se procesan inmediatamente en tu dispositivo usando Google ML Kit offline para extraer el texto y NO se almacenan permanentemente ni se envían a ningún servidor externo. Una vez procesado el texto, la imagen se descarta automáticamente
  • Audio para reconocimiento de voz: Cuando usas funciones de voz a texto, el audio se procesa localmente usando tecnologías como Vosk API y sherpa-onnx. Las grabaciones NO se almacenan y el procesamiento es completamente offline
  • Preferencias de voz: Configuraciones de TTS (texto a voz) y STT (voz a texto) para personalizar la experiencia
  • Modelos de IA descargados: Los modelos de inteligencia artificial se almacenan localmente en tu dispositivo para funcionamiento offline
  • Datos de uso básicos (opcional): Solo para Gemini Live - si eliges usar servicios online, se envían datos según las políticas de Google

2.2 Datos que NO recopilamos:

  • No recopilamos información personal identificable (nombre, dirección, teléfono) excepto lo que tú escribas voluntariamente en las conversaciones
  • No accedemos a tus contactos
  • No enviamos conversaciones a servidores externos (excepto si usas Gemini Live voluntariamente)
  • No utilizamos servicios de analítica de terceros
  • No recopilamos datos de ubicación
  • No utilizamos ID de publicidad ni mostramos anuncios
  • No accedemos a otras aplicaciones o datos del sistema
  • No monitoreamos tu actividad fuera de la aplicación

3. Cómo Usamos los Datos

  • Funcionalidad del chatbot: Para proporcionar respuestas de IA y mantener el contexto de la conversación usando modelos locales o Gemini Live (opcional)
  • Mejora de la experiencia: Para recordar tus preferencias y configuraciones
  • Moderación de contenido: Para procesar reportes de contenido inapropiado y mejorar los filtros
  • Procesamiento local: La mayoría de funciones de IA funcionan completamente offline en tu dispositivo
  • Procesamiento de imágenes con OCR: Las imágenes capturadas con la cámara se procesan instantáneamente usando tecnología de reconocimiento óptico de caracteres (OCR) de Google ML Kit, que funciona completamente offline en tu dispositivo. No se almacenan las imágenes ni se envían a servidores externos
  • Reconocimiento y síntesis de voz: Procesamiento local de audio usando Vosk API (STT) y sherpa-onnx (TTS) para convertir voz a texto y texto a voz sin conexión a internet
  • Gestión de modelos: Descarga y almacenamiento local de modelos de IA para funcionamiento offline
  • Servicios opcionales online: Solo si eliges usar Gemini Live, algunos datos se envían a Google según sus políticas
  • Notificaciones: Para informarte sobre el progreso de descargas y estado de la aplicación

4. Almacenamiento y Seguridad

  • Almacenamiento local: Todos los datos se almacenan únicamente en tu dispositivo
  • Sin transmisión: No se envían datos a servidores externos
  • Cifrado: Los datos sensibles se almacenan de forma segura usando las APIs de Android
  • Control del usuario: Puedes eliminar todos los datos desinstalando la aplicación

5. Compartición de Datos

Principio general: No compartimos datos con terceros.

5.1 Modo Offline (Predeterminado):

  • La aplicación funciona completamente offline
  • No se transmite información a servicios externos
  • Todos los datos permanecen en tu dispositivo

5.2 Gemini Live (Opcional):

  • Si decides usar la función Gemini Live, tus mensajes se envían a Google Gemini API
  • Esta función es completamente opcional y debe ser activada manualmente
  • Los datos enviados a Gemini están sujetos a las políticas de privacidad de Google
  • Puedes desactivar esta función en cualquier momento
  • Siempre se te informa claramente cuando usas servicios online

5.3 Tecnologías de Terceros (Solo Locales):

  • Google ML Kit: OCR offline - no envía datos a Google
  • Vosk API: STT offline - no envía datos a servidores
  • sherpa-onnx: TTS offline - procesamiento local únicamente
  • llama.cpp: Inferencia de IA local - sin conexión externa

6. Derechos del Usuario

Tienes derecho a:

  • Acceder: Ver todos los datos almacenados en la aplicación
  • Modificar: Editar o eliminar conversaciones y configuraciones
  • Eliminar: Borrar todos los datos desinstalando la aplicación
  • Exportar: Utilizar la función de compartir para exportar conversaciones

7. Eliminación de Datos

  • Puedes eliminar conversaciones individuales desde la aplicación
  • Puedes restablecer todas las configuraciones desde el menú de ajustes
  • La desinstalación de la aplicación elimina automáticamente todos los datos
  • No mantenemos copias de respaldo de tus datos

8. Menores de Edad

La aplicación está diseñada para usuarios mayores de 13 años. No recopilamos intencionalmente datos de menores de 13 años. Si te das cuenta de que un menor ha proporcionado datos, contáctanos para que podamos eliminarlos.

9. Cambios en la Política

Podemos actualizar esta política ocasionalmente. Los cambios se notificarán a través de la aplicación y se publicarán en esta página. El uso continuado de la aplicación constituye la aceptación de los cambios.

11. Declaración Específica de Permisos Requeridos por Google Play

En cumplimiento con las políticas de Google Play Store, declaramos específicamente el uso de permisos sensibles:

📷 Permiso de Cámara (android.permission.CAMERA)

Uso específico: Reconocimiento Óptico de Caracteres (OCR) para extraer texto de imágenes

Procesamiento: Completamente local usando Google ML Kit offline

Almacenamiento: Las imágenes NO se guardan, se procesan y descartan inmediatamente

Transmisión: NO se envían imágenes ni datos a servidores externos

Control del usuario: Funcional opcional - la app funciona sin este permiso

🎤 Permiso de Micrófono (android.permission.RECORD_AUDIO)

Uso específico: Reconocimiento de voz local y funciones de texto a voz

Tecnologías utilizadas: Vosk API y k2-fsa/sherpa-onnx (offline)

Procesamiento: Completamente local en el dispositivo

Almacenamiento: NO se guardan grabaciones de audio

Transmisión: NO se envían datos de audio a servidores externos

🔊 Permiso de Modificación de Audio (android.permission.MODIFY_AUDIO_SETTINGS)

Uso específico: Optimizar la calidad del audio para funciones TTS (texto a voz)

Funcionalidad: Ajustar configuraciones de audio solo durante la reproducción de voz

Alcance: Solo afecta la experiencia de audio de la aplicación

🌐 Permiso de Internet (android.permission.INTERNET)

Uso específico: Descarga inicial de modelos de IA únicamente

Funcionamiento post-descarga: La aplicación funciona completamente offline

Sin analítica: NO se envían datos de uso o telemetría

Nota importante: También incluye integración opcional con Gemini Live API para usuarios que decidan usar servicios online

📡 Permisos de Red (android.permission.ACCESS_NETWORK_STATE, ACCESS_WIFI_STATE)

Uso específico: Verificar conectividad para descargar modelos y para Gemini Live

Funcionamiento: Solo para informar al usuario del estado de conexión

No recopilación: NO se recopilan datos sobre tu red

🖼️ Permiso de Lectura de Imágenes (android.permission.READ_MEDIA_IMAGES)

Uso específico: Permite al usuario seleccionar imágenes de su galería para procesamiento OCR

Acceso limitado: Solo lee imágenes seleccionadas por el usuario mediante el selector de archivos del sistema

Procesamiento: Las imágenes se procesan localmente para extraer texto usando OCR offline

Almacenamiento: NO se copian ni guardan las imágenes, solo se procesan temporalmente

Transmisión: NO se envían imágenes a servidores externos

Control del usuario: El usuario controla qué imágenes seleccionar - funcionalidad opcional

🔔 Permisos de Notificaciones (android.permission.POST_NOTIFICATIONS)

Uso específico: Mostrar notificaciones de progreso durante la descarga de modelos

Funcionalidad: Informar al usuario sobre el estado de las operaciones de la app

Control del usuario: Puede denegarse en configuración del sistema

⚙️ Servicios en Primer Plano (android.permission.FOREGROUND_SERVICE)

Uso específico: Mantener procesos de IA funcionando cuando la app está en segundo plano

Propósito: Asegurar que las conversaciones largas no se interrumpan

Transparencia: Se muestra una notificación persistente cuando está activo

📂 Permisos de Almacenamiento (android.permission.WRITE_EXTERNAL_STORAGE - Solo Android ≤10)

Uso específico: Almacenar modelos de IA descargados en dispositivos antiguos

Ámbito limitado: Solo en la carpeta privada de la aplicación

Android 11+: Utiliza almacenamiento con ámbito limitado (más seguro)

⏰ Permisos de Alarmas Exactas (android.permission.SCHEDULE_EXACT_ALARM)

Uso específico: Programar recordatorios y notificaciones de salud (si se usa esa función)

Funcionalidad: Asegurar que las notificaciones se muestren en el momento exacto programado

Control del usuario: Solo se activa si el usuario configura recordatorios

🔄 Permiso de Arranque Automático (android.permission.RECEIVE_BOOT_COMPLETED)

Uso específico: Restaurar recordatorios programados después de reiniciar el dispositivo

Funcionalidad: Solo reactiva funciones que el usuario había configurado previamente

Privacidad: NO se ejecuta ningún código sin conocimiento del usuario

12. Permisos de la Aplicación y Su Uso Específico

La aplicación solicita los siguientes permisos y los utiliza de la siguiente manera:

📱 Permisos Principales:

  • Permiso de Cámara (android.permission.CAMERA):
    • Propósito específico: Únicamente para la función de OCR (Reconocimiento Óptico de Caracteres)
    • Cuándo se activa: Solo cuando presionas el botón de "Escanear texto" o "OCR" en la aplicación
    • Qué hace: Captura una fotografía del texto que deseas extraer (documentos, pantallas, etc.)
    • Procesamiento: La imagen se procesa INMEDIATAMENTE en tu dispositivo usando Google ML Kit offline
    • Almacenamiento: La imagen se descarta automáticamente después de extraer el texto. NO se guarda en tu dispositivo ni se envía a internet
    • Control del usuario: Puedes denegar este permiso y la aplicación seguirá funcionando normalmente, solo sin la función de OCR
  • Permiso de Micrófono (android.permission.RECORD_AUDIO):
    • Propósito: Funciones de reconocimiento de voz (STT) y IA Live
    • Tecnología: Vosk API para procesamiento local offline
    • Almacenamiento: NO se guardan grabaciones - procesamiento inmediato
    • Control: Opcional - funciona sin este permiso
  • Permiso de Internet (android.permission.INTERNET):
    • Uso principal: Descarga inicial de modelos de IA
    • Uso secundario: Gemini Live (opcional) para usuarios que quieren servicios online
    • Post-descarga: La mayoría de funciones trabajan offline
    • Transparencia: Se indica claramente cuando se usan servicios online
  • Lectura de Imágenes (android.permission.READ_MEDIA_IMAGES):
    • Propósito: Seleccionar imágenes de galería para OCR
    • Acceso: Solo a imágenes seleccionadas manualmente por el usuario
    • Procesamiento: Local usando ML Kit offline
    • Almacenamiento: NO se copian ni guardan las imágenes

⚙️ Permisos del Sistema:

  • Modificar Configuración de Audio: Optimizar calidad de voz para TTS
  • Servicios en Primer Plano: Mantener IA funcionando en segundo plano
  • Notificaciones: Informar progreso de descargas y estado de la app
  • Estado de Red: Verificar conectividad solo para descargas necesarias
  • Almacenamiento: Guardar modelos de IA y configuraciones localmente
  • Alarmas Exactas: Recordatorios de salud (si se configura)
  • Arranque Automático: Restaurar recordatorios después de reinicio

Importante: Todos los permisos se utilizan únicamente para el funcionamiento local de la aplicación. Excepto por Gemini Live (opcional), NO se envían datos a servidores externos bajo ninguna circunstancia.
